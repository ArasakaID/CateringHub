## Route & Backend Plan

### Database
Sudah ada di `menus`:
- `id`, `catering_id`, `category_id`, `name`, `description`, `price`, `image`, `is_available`

**Yang perlu ditambah:**
- [ ] Kolom `options` (JSON) di `menus` вҖ” untuk menyimpan opsi pilihan:
  ```json
  [{"name": "Nasi hangat"}, {"name": "telur puyuh (rebus dan kupas)"}]
  ```

### Route Baru
```php
Route::get('/menu/{id}', [MenuController::class, 'show'])->name('menu.show');
```

### MenuController
- `show($id)` вҖ” ambil satu menu dengan relasi `catering`
- Kembalikan Inertia render `DetailProduk` dengan data:
  - `menu` вҖ” detail menu
  - `catering` вҖ” parent catering (untuk nama + logo)
  - `relatedMenus` вҖ” menu lain dari catering yang sama (opsional)

### Page Baru
- `resources/js/Pages/DetailProduk.jsx` вҖ” Halaman Detail Produk Inertia

### Link dari Halaman Lain
